Завдання:

Для проекту з країнами та містами додайте інтерфейс IEmailSender. Цей інтерфейс має методи void Send(string title, string text) 
та асинхронну версію Task SendAsync(string title, string text, CancellationToken calcellationToken). Додайте відправлення листів 
по пошті через цей інтерфейс для методів при додаванні, видаленні або редагуванні країн/міст/готелей. 

Оскільки для реалізації розсилки по пошті необхідний SMTP сервер, придумайте спосіб як запустити проект і мати змогу дивитися 
який текст листів приходить, без використання серверу. Функціонал відправки листів має бути збережений, можна змінювати тільки 
клас що імплементує інтерфейс IEmailSender.
