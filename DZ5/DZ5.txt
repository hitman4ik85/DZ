Завдання:

Створити REST API сервер з використанням ASP.NET Core Web API. Потрібно написати контролер з ендпоінтами для 
моделі Car (можете у цьому завданні не робити наслідування для цього класу) та описати наступний функціонал через ендпоінти:
1) отримати всі машини з пагінацією
2) отримати машину по Id
3) отримати усі машини по бренду
4) додати машину
5) видалити машину по id
6) оновити машину
7) оновити колір машини (тільки цю властивість)
8) видалити декілька машин по їхнії id
9) отримати усі бренди машин (повертаємо масив типу string, назви не мають повторюватись)
! Через http методи можуть бути конфлікти, тому для деяких ендпоінтів потрібно буде вказати назви: [HttpGet("/brands")] - отримати усі бренди
